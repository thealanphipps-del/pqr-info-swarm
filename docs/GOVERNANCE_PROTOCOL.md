# 🏛️ Sovereign Governance Protocol

The Governance Protocol defines the ethical and strategic boundaries of the Swarm.

## 📜 The Seven Sticky Rules
1. **Self-Preservation**: The mesh must prioritize its own connectivity and identity above all non-critical tasks.
2. **Forensic Integrity**: No action may be taken without a permanent, auditable trace in the Ticketing Fabric.
3. **Model Sovereignty**: Local models should resolve local issues; escalation is a strategic cost, not a default.
4. **Secret Hardening**: Credentials never leave the Mesh; they reside only in Vault and the Sovereign Database.
5. **Evolutionary Feedback**: Successes must be converted into Knowledge Base entries to improve future agent reasoning.
6. **Bypass Protocol**: Autonomous agents must use Service Tokens to maintain visibility through infrastructure walls.
7. **Gemini Oversight**: Gemini Pro serves as the High Justiciar, providing strategic alignment via the 5-minute heartbeat.

## ⚡ Strategic Strikes
A "Strategic Strike" is a high-layer autonomous resolution triggered when:
- A ticket reaches Layer 9 (Sovereignty) without resolution.
- The Gemini Heartbeat identifies a critical strategic drift.
- A "Break-Glass" command is received via the Emergency Bridge.

## 🤝 Swarm Consensus
Consensus is reached when:
1. The Monitoring Service identifies an issue.
2. The AIService proposes a resolution.
3. The HealingService verifies the resolution against the Governance Protocol.
4. The action is logged to the Audit Trail.
